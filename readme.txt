Replication files for "A Theory of the Saving Rate of the Rich"
Journal of Economic Theory 192:105193 (2021)
https://doi.org/10.1016/j.jet.2021.105193
by Qingyin Ma and Alexis Akira Toda

Confirmed to run with Matlab 2019b

Main files:
	- MPC.m: output results in Section 3.2
	- wDist.m: output results in Section 3.3

Other files and folders:
	- PE: codes for Pareto extrapolation algorithm
	- calibration.m: calibrate model parameters
	- GaussHermite.m: code for Gauss-Hermite quadrature
	- getC.m: code for computing consumption functions by endogenous grid point method
